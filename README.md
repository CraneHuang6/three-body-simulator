# 三体实验室

一个以《三体》为灵感的 Mac 桌面应用。

你可以把它理解成一个可直接观看、也可自己动手调参数的科幻互动作品。

## 下载

请直接前往发布页下载：

[下载最新版本](https://github.com/CraneHuang6/three-body-lab/releases/latest)

建议普通用户优先下载：

- `三体实验室-0.1.0-arm64-mac.dmg`

如果 `dmg` 无法打开，也可以尝试：

- `三体实验室-0.1.0-arm64-mac.zip`

## 运行要求

当前发布版本适用于：

- macOS
- Apple Silicon 芯片（M1 / M2 / M3 / M4）

暂不面向 Windows 和 Intel Mac 普通用户提供开箱即用说明。

## 安装步骤

1. 在发布页下载 `dmg` 文件。
2. 双击打开下载好的文件。
3. 把 `三体实验室` 拖入 `Applications` 文件夹。
4. 打开“应用程序”，找到 `三体实验室` 并启动。

## 第一次打不开怎么办

如果 macOS 提示“无法打开”或“来自未知开发者”，按下面做：

1. 在“应用程序”里找到 `三体实验室`
2. 按住 `control` 键点击应用，选择“打开”
3. 再点一次“打开”

通常第一次这样处理后，后面就能正常启动。

## 打开后怎么玩

应用首页有两个入口：

- `观演模式`：直接观看八大天象演示
- `实验室模式`：手动调整 4 个天体参数，观察轨迹和碰撞变化

如果你只是第一次体验，建议先看：

1. `观演模式`
2. 再切到 `实验室模式`

这样更容易理解这个项目原本的演示效果和可交互部分。
